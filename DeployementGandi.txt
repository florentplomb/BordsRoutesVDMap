Bonjour,
j'étais aussi dans le même cas, et en faisant plusieurs essais, j'ai fini
par trouver la solution :
tout d'abord faire un git push sur le dépot gandi
git push gandi master
(où "gandi" est le remote repository pour le serveur gandi, souvent appelé
"origin")
et ensuite :
ssh XXXXX@git.YYY.gpaas.net 'deploy default.git'
où XXX sont les identifiant (série de chifres) et YYY la localisation du
serveur (dc0, dc1...)
Ce qui copie effectivement les fichiers et installe les dépendances listés
dans le fichier "package.json"
J'espère que c'est clair ;)

À partir de l’adresse <http://groups.gandi.net/fr/topic/gandi.fr.hebergement.simple/36542>
